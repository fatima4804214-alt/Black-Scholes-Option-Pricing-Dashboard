# 📈 Black-Scholes Option Pricing Dashboard

A live stock options pricing dashboard built with Python and Streamlit.

## Features
- Live stock data via Yahoo Finance
- Black-Scholes call & put option pricing
- Annualized volatility calculation
- Interactive 6-month price chart

## How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Files
- `app.py` — Main Streamlit dashboard
- `black_scholes.py` — Black-Scholes pricing model
- `volatility.py` — Volatility calculation
- `requirements.txt` — Python dependencies
